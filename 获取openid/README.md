#### 安装教程

1. clone本项目到本地
2. 使用idea或其他开发工具打开本项目
3. 等待maven依赖装完
4. 若依赖安装不上，请到项目设置里将jdk改成本机jdk，然后将maven设置里依赖的jdk也改为本机的jdk
